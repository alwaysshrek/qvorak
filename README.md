This extension adds keyboard shortcuts to your Quora.

## Shortcuts 
###Global:
* ? - show help

###Dashboard:
* / - search
* f - Follow/Unfollow a question
* Alt+Q - Add a question
* Alt+P - Add a post
* Alt+H - Go to Home
* Alt+U - Go to Profile
* Alt+N - Go to Notifications
* Alt+I - Go to Inbox
* Alt+B - Go to Boards
* Alt+T - Go to Topics
